O que é Indexação em Banco de Dados?

A indexação é uma técnica usada em bancos de dados para melhorar a eficiência das operações de busca. Um índice age como um mapa que ajuda a localizar dados sem precisar pesquisar por todas as entradas da tabela.

Conceito de Índices

Índices são estruturas de dados que permitem o acesso mais rápido aos registros de uma tabela. Existem diferentes tipos de índices, cada um adequado para diferentes tipos de consultas.

B-tree

O B-tree (árvore-B) é o tipo de índice mais comum em bancos de dados relacionais. Ele organiza os dados de forma hierárquica, permitindo a busca rápida, inserção e remoção de elementos.

Hash

Índices de hash são baseados em funções de hash, que convertem valores em uma posição específica de memória. Eles são altamente eficientes para consultas de igualdade.

Bitmap

O índice de bitmap é utilizado para colunas com poucos valores distintos (baixa cardinalidade), como colunas booleanas ou de status. Ele armazena os dados como bits, otimizando o espaço e tempo de busca.
